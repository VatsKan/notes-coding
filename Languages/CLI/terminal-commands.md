# General terminal commands

* opens a new terminal window 
    - with the same working directory
    ```open -a Terminal "`pwd`"```
    - in the home directory
    ```open -a Terminal ~```
    - in general
    ```open -a Terminal <filepath>```

* to list all files in a directory (including hidden files)
    ```ls -a```

* To look up: 
    ```echo```
    ```cat```
    ```nano```
    ```z```
    ```top```
    ```sudo```
